User Guide for using the calculator

  • Limits

      General form : lim(function, variable, limit, side)

      *The side parameter is optional. Use '+' and '-' when approaching the limit from right and left respectively.

      Example : lim(cosx/x, x, 0, '-')

      This will evaluate left hand limit of the function 'cosx/x' when x→0-

      Example : lim(sinx/x, x, 0)

      This will evaluate the limit of the function 'sinx/x' when x→0.


  • Differentiation

      General form : diff(function, variable, nth derivative)

      Example : diff(3x^2 - xy, x, 1)

      This will evaluate the first derivative of the function '3x^2 - xy' with respect to 'x'.

      Example : diff(2xy^2 - x^y, x, 4)

      This will evaluate the fourth derivative of the function '2xy^2 - x^y' with respect to 'x'.


  • Integration

    • Single Integration

        General form : ∫ (function, variable, lower limit, upper limit)

        *The lower and upper limit parameters are optional.

        Example : ∫ (2x^3 - 3*sin4x, x, -1, 1)

        This will integrate the function '2x^3 - 3*sin4x' with respect to 'x' under the limit -1 to 1.

        Example : ∫ (x^7 - 3^x, x)

        This will integrate the function 'x^7 - 3^x' with respect to 'x'.

    • Double Integration

        General form : ∫ (function, (variable-1, lower limit, upper limit), (variable-2, lower limit, upper limit))

        *The lower and upper limit parameters are optional.

        Example : ∫ (x^4 - 3^y, (x, -2, 2), (y, 3, 6))

        This will integrate the function 'x^4 -3^y' first with respect to 'x' under the limit -2 to 2 and then again integrate with respect to 'y' in the range 3 to 6.

        Example : ∫ (7^x - log(xy), (x), (y))

        This will integrate the function '7^x - log(xy)' with first with respect to 'x' then with respect to 'y'.

    • Triple Integration

        General form : ∫ (function, (variable-1, lower limit, upper limit), (variable-2, lower limit, upper limit), (variable-3, lower limit, upper limit))

        *The lower and upper limit parameters are optional.

        Example : ∫ (x^2 + y^2 - z^2, (x, -1, 1), (y, -x, x), (z, y, 2*y))

        This will integrate the function 'x^2 + y^2 - z^2' first with respect to 'x' under the limit -1 to 1 then with respect to 'y' under the limit -x to x and then with respect to 'z' under the limit y to 2y.

        Example : ∫ (log(3x-2) + siny^2 - 2^z, (x), (y), (z))

        This will integrate the function 'log(3x-2) + siny^2 - 2^z' first with respect to 'x' then with respect to 'y' and then with respect to 'z'.


  • Beta and Gamma functions

    • Beta function

        General form : β(m,n)

        Example : β(5/2, 3/2)

        This will calculate the value of beta function when m = 5/2 and n = 3/2.

    • Gamma function

        General form : Γ(n)

        Example : Γ(5)

        This will calculate the gamma of 5.

  • Transforms

    • Laplace Transform

        General form : L(function)

        Example : L(t^5 - t^2)

        This will perform Laplace Transform on the function 't^5 - t^2'.

    • Inverse Laplace Transform

        General form : L_inv(function)

        Example : L_inv(s^-3 - s^2)

        This will perform the Inverse Laplace Transform on the function 's^-3 - s^2'.

    • Fourier Transform

        General form : fourier_series(function, (variable, lower limit, upper limit))

        Example : fourier_series(x^2, (x, -1, 1))

        This will return the Fourier Series of the function 'x^2' with respect to x under the limit -1 to 1.
